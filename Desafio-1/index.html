O que são Clients e Servers?

Voltar

O termo client e server, são termos utilizado dentro da arquitetura de rede, ambos são termos aplicados em uma relação de dois ou mais individuos presentes. O client, é o termo utilizado para se referir ao eletronico que detém uma interface gráfica que possibilita a navegação ou acesso a dados e aplicativos, ja o server, são computadores que detém como objetivo, gerir e fornecer dados para o solicitantes (client), parar que o mesmo possa consumir a informação que lhe interessa.

O que são tags?

Voltar

Segundo Semrush Blog:

"Tags HTML são instruções que informam ao navegador como formatar o texto. Você pode usar tags para definir itálico, quebras de linha, objetos, bullet points e muito mais."

Alguns exemplos de tags são: tag de paragrafro (p), tag de título (h1 até h6), tag para colocar em negrito (strong), tag para colocar em italico (i) e etc.

O que são atributos?

Voltar

Os atributos, em suma, são forma de passar in formações para as tags para que as mesmas se comportem conforme necessitamos, um exemplo classico, é a tag input, essa tag cria um campo para preencimento e o seu comportamento muda conforme atribuimos atributos, por exemplos, se o campo input for receber um número, informamos que o mesmo é do "type" number, desta forma, o mesmo só aceitará números, caso o mesmo receba um texto, atribuímos o "type" como text.

html e sua estrutura básica

Voltar

Para entender a estrutura do html, vamos precisar entender as tags que aparecem em seu corpo, desta forma, use a imagem a seguir, para entender o seu comportamento, conceito e intrução que passa no código.

A estrura básica do html é composta por 3 tags, sendo elas:

  • html
  • head
  • body

O que é a tag html?

A tag html é representa a raiz do documento, serve com um container que engloba todos os outros elementos html.

O que é a tag head?

Compreende as informações do documento que serão interpretadas pelo navegador (metadados). Como por exemplo, título do documento, links para folhas de estilo etc.

O que é a tag body?

É onde fica todo o conteúdo de texto, imagem e vídeos, em que o usuário vê e interage, nele os conteúdos são estruturados pelas demais tags do html.
